## Connecting to Google Analytics & Search Console

### Step 1 — Update the website URL in GBP
Instead of the plain URL, use this UTM-tagged version so GA4 can track clicks from your GBP listing separately:
```
https://dandelionwishescounseling.com/?utm_source=google_business_profile&utm_medium=organic
```

### Step 2 — Set up Google Search Console
1. Go to [search.google.com/search-console](https://search.google.com/search-console)
2. Add `dandelionwishescounseling.com` as a **Domain** property
3. Verify ownership — choose the **Google Analytics** method (it detects your existing GA4 tag automatically, no code changes needed)
4. Once verified, submit your sitemap: `https://dandelionwishescounseling.com/sitemap.xml`

### Step 3 — Link Search Console to GA4
1. In GA4: **Admin → Property → Product Links → Search Console Links → Link**
2. Select your Search Console property
3. Done — you'll now see search query data inside GA4 reports
