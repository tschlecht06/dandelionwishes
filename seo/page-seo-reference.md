## URL Slugs

Set these in **Pages > [Page] > Settings > General > URL Slug**.

| Page | Slug |
|------|------|
| Home | `/` (default) |
| About | `about` |
| Services & Fees | `services` |
| FAQ | `faq` |
| Blog | `blog` |
| Contact | `contact` |

Use lowercase, no trailing slashes. Do not use auto-generated slugs from the page title (e.g., avoid `services-fees`).
